Struct couchdb::path::DesignDocumentId [−] [src]

pub struct DesignDocumentId(_);

DesignDocumentId comprises URL path segments that, together, identify a design document.

For example, given the document path /db/_design/doc, the design document id is _design/doc.

DesignDocumentId is a special form of DocumentId. All design document ids are document ids, but not all document ids are design document ids.

For more information about path-related types, see the module-level documentation.
