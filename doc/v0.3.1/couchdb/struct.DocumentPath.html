pub struct DocumentPath(_, _);

Document path—i.e., a database path paired with a document id.

A document path pairs a DatabasePath with a DocumentId—e.g., the db and docid parts in the HTTP request GET http://example.com:5984/db/docid.

DocumentPath provides additional type-safety over working with raw strings.
